# Hywood – Real Wood Engineered Parquet
**Ter Hürne General Price List · Edition 03-2025 · Valid from 01.07.2025**
**Source: pricelist.pdf, pages 30–36**

## Technical Properties

- **Construction:** Multi-layer engineered wood
  - Layer 1a: Nature Protec EVO surface sealant (lacquer)
  - Layer 1b: Nature Protec OLIO surface sealant (oil)
  - Layer 2: Design layer (real wood)
  - Layer 3: Wood powder layer (wood powder + high-grade resins)
  - Layer 4: Aqua-Seal baseboard (extremely high-density wood fibres)
  - Layer 5: Aqua-Seal profiling "CLICKitEASY Pro"
  - Layer 6: Wood powder layer (wood powder + high-grade resins) or stabilising layer (real wood)
- **Connection systems:**
  - **CLICKitEASY Pro** – for plank and extra-wide plank formats; floating and glue-down installation
  - **CLICKitEASY HB** – for herringbone formats; floating and glue-down installation
- **Underfloor heating:** Suitable for water-based and electric systems. For electric surface heating: gentle warm-up technology + automatic dew point regulation required. Reverse cycle cooling systems must have automatic dew point regulation.
- **Wetroom suitability:** Planks and extra-wide planks: • (suitable). Herringbone: - (not suitable)
- **Certifications:** U.S. EPA TSCA Title VI & CARB P2 compliant | W10-12 | ID 0322-11806-009 | www.realwood.eu
- **Note:** Natural colour and structure differences are possible.
